# HackerTool
## Note: HackerTool from 2.0 to 2.32 will be permanently unusable on 6/28/2024
HackerTool is a powerful program writed in Batch and PowerShell with Easy GUI for things like
- Geolocation
- Ping
- Proxy Check
- PaySafeCard Generator
- Random Number
- String Search
- Domain Check
- API Connection
